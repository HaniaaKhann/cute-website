# 💕 Cute Interactive Yes/No Webpage

A fun, aesthetic, soft-girl styled interactive webpage where the user is asked a playful question. The "No" button escapes when hovered, and the "Yes" button grows each time the user tries to click "No".

---

## 🌸 Features

- 💖 Interactive Yes / No buttons  
- 😈 "No" button runs away on hover  
- 📈 "Yes" button grows with each interaction  
- 🎀 Soft aesthetic UI design  
- 🌸 Full-screen background image  
- 💌 Floating heart animations  
- ✨ Smooth transitions and playful interactions  

---

## 🛠️ Built With

- HTML  
- CSS  
- JavaScript  
- Custom font (PIXIE)  
- DOM manipulation
